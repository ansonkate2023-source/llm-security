# AI Red Teaming 工具與框架

**分類：** tool
**最後更新：** 2026-04-06

## 開源工具

### PyRIT（Python Risk Identification Tool）
- **開發者：** Microsoft
- **用途：** 生成式 AI 系統風險識別
- **GitHub：** https://github.com/Azure/PyRIT
- **特點：** 自動化紅隊測試、多種攻擊策略

### Garak（Generative AI Red-teaming and Assessment Kit）
- **開發者：** NVIDIA
- **用途：** LLM 漏洞掃描
- **GitHub：** https://github.com/NVIDIA/garak
- **特點：** 廣泛的漏洞偵測、可擴展架構

### DeepTeam
- **開發者：** Confident AI
- **用途：** LLM 紅隊滲透測試框架
- **GitHub：** https://github.com/confident-ai/deepteam
- **特點：** 簡單易用、開源

### AI Red Teaming Guide
- **GitHub：** https://github.com/requie/AI-Red-Teaming-Guide
- **用途：** AI 系統對抗測試與安全評估的綜合指南
