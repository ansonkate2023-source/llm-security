# AI 可觀測性與監控工具

**分類：** tool
**最後更新：** 2026-04-06

## 概述

2026 年 AI 可觀測性已從 prompt 除錯進化為安全且高效地營運 LLM 系統的基礎能力。平台提供追蹤、評估、告警與治理的綜合解決方案。

## 核心功能

### 追蹤（Tracing）
- Prompt → Response 完整追蹤
- Agent 工作流程可視化
- Token 使用量與延遲監控

### 評估（Evaluation）
- 輸出品質與準確性評估
- 幻覺偵測
- 安全護欄有效性測試

### 安全監控
- Prompt injection 偵測
- Jailbreak 嘗試告警
- PII 洩露監控
- 異常行為偵測

### 治理（Governance）
- 合規審計追蹤
- 成本監控與優化
- 模型效能基線
